# Cisco Networking Labs
A collection of labs from my Cisco 2 coursework at MATC, combined with physical configurations from my personal homelab.

## 🏠 Homelab Integration
This folder includes a mix of academic simulations and real-world configurations from my **Cisco 780 router**, which I manage via console cable to test routing protocols and security settings in a physical environment.

## 🛠️ Featured Labs
* **[DHCPv4 Implementation](./CesarV7.2.10-packet-tracer---configure-dhcpv4.pka)**
  * **Goal:** Configure a Cisco router as a DHCP server to automatically assign IP addresses to multiple subnets.
  * **Key Skills:** DHCP pools, excluded addresses, and relay agents.
